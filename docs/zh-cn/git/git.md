### 设置用户签名
`git config --global user.name jieni`

`git config --global user.email 759835420@qq.com`

### 初始化本地库
`git init`

### 查看本地库状态
`git status`

### 添加到暂存区
`git add 文件名称`

### 删除暂存区的文件
`git rm --cached 文件名称`

### 提交本地库
`git commit -m "备注信息" 文件名称`

### 查看历史版本信息
`git reflog`

`git log `

### 版本穿梭(指向历史版本测试)
`git reset --hard 版本号`
